# get-first-words
Get first word from sentence

## Installation
```npm
  yarn add get-first-words
  or
  npm install get-first-words
```
## Usage
```javascript
const gfw = require('get-first-words');

var sentence = "Get First Words";
console.log(gfw(sentence)); //return "Get"
```

## Dev
`node index.js`

## Run Test
`npm test`
